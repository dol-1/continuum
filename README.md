## 🇬🇧 English

**Continuum** is a personal "second brain" integration layer that unifies MemPalace, Page Assist, local Gemma 4 26B, n8n, and a Telegram bot into one living memory system.

### Why?
Modern AI tools are brilliant but forgetful. Continuum captures conversations, analyzes them locally with Gemma, and builds your personal, searchable, permanent memory layer.

### Core Features
- 🔒 **Local-first** — your data never leaves your server
- 🌐 **Multi-AI capture** — Claude.ai, ChatGPT, Gemini in one place
- 🧬 **Memory Consolidation Engine** — Anthropic Auto Dream pattern
- 🇬🇪 **Bilingual** — Georgian + English first-class
- 📱 **Telegram integration** — capture from anywhere
- 👁️ **Obsidian-compatible** — optional desktop viewer
